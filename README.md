An ultra-lightweight, zero-dependency relational database engine engineered for high efficiency across ESP32 microcontrollers, Linux, and Windows hosts.

Key Features:

Binary Storage: Minimizes disk space and flash wear through a direct-to-disk binary formatting system.

Relational Integrity: Implements native constraints, foreign keys, and cascaded delete/drop workflows.

Indexed Queries: Speeds up lookups and conditional sorting using modular index structures.

Cross-Platform VFS Integration: Smoothly handles hardware-specific abstraction layers, including ESP32's LittleFS and standard desktop filesystems.
